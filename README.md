<h1>FUNCIONAMENTO DE UM RETIFICADOR<h1>

<p>Normalmente, os diodos retificadores são ligados a uma fonte de energia e desta forma é gerada uma tensão elétrica, que pode ser direta ou reversa. Caso os terminais positivos dos diodos retificadores sejam ligados aos terminais positivos da fonte de energia, ocorre um processo conhecido como polarização direta.<p>
